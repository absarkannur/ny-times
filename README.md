#
# React Redux + Vite

## Getting Started

Install the project dependencies.

```bash
yarn install
```
### or
```bash
npm install
```
#

First, run the development server:

```bash
yarn dev
```
### or
```bash
npm run dev
```
#

Run the React Testing:

```bash
yarn test
```
### or
```bash
npm run test
```
#

Run the Cypress Testing

```bash
yarn cypress
```
### or
```bash
npm run cypress

# then use the article.cy.js

```
