#
# React Redux + Vite

## Getting Started

Install the project dependencies.

```bash
yarn install
# or
npm install
```

First, run the development server:

```bash
yarn dev
# or
npm run dev

```

Run the React Testing:

```bash
yarn test
# or
npm run test

```

Run the Cypress Testing

```bash
yarn cypress
# or
npm run cypress

# then use the article.cy.js

```
